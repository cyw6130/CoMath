研究兴趣

Khovanov Homology

khovanov homology 是琼斯多项式的范畴化, 从 khovanov homology的基本定义出发, 发展出了丰富的内容.

  • 首先, 它是纽结不变量, 可以区分纽结.
  • 通过形变的 khovanov homology (Lee homology), 我们可以定义s-invariant, 它可以用来限制纽结的亏格, 从而可以证明 \mathbb{R}^4上存在怪异的光滑结构.
  • 从 cube 出发, 我们可以定义一个广义空间(谱:spectrum), 它是纽结不变量(可以用来区分 khovanov homology 无法区分的纽结), 这套理论将纽结和稳定同伦论联系起来.
  • 近些年, skein lasagna module 定义了四维流形的光滑不变量, Qiuyu Ren 等人首次利用这个不变量区分了光滑四维流形的怪异对(两个同胚却不微分同胚的流形).
