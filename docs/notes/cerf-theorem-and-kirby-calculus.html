不特殊说明, 我们总考虑连通, 定向, 光滑, 紧的, (可能) 不闭的情形.

    • 但是: 对于任意一个 Kirby diagram, 我们仅考虑 0,1,2 handle. 它给出的带边 4 mfld 的边界未必是 S^1\times S^2 的边界连通和, 故它未必能够通过粘接 3,4 handle 得到闭的 4 mfld. 所以, 未必任意的 Kirby diagram 都能给出闭的 4 mfld.

    • handle 分解: (X,\partial X) 的 handle 分解是一些 handle h^k_i, 满足 X=\partial X\times I+\bigcup_i h_i^k.

4 Results in Dimension 3

4.1 Recall: Surgery

  • Surgery:
    • 我们将 l-surgery 定义为从M^m上挖去一个S^l\times D^{m-l}(这个时候挖去的部分会裸露出来一个S^l\times S^{m-l-1}), 然后沿着裸露出来的新的边界粘回去一个 D^{l+1}\times S^{m-l-1}, 其中边界上的粘接映射就是 identity.
    • 注意, 当 m=3,M=S^3,l=1时, 在M^m上选取一个S^l\times D^{m-l}, 就对应 S^3 上的一个 framed knot, 我们取 Seifert framing 的话, 它就会被一个整数标记. 因此, 当考虑 3 mfld 的时, 我们也将这种 Surgery 称为整的 Surgery.
  • Surgery 与粘接 handle:
    • 我们在流形 W^{n+1}(或者把它看作配边(W;W_-,W_+)) 粘接 k+1 handleD^{k+1}\times D^{n-k}(当做配边看的时候, 就是在上边界W_+上粘接), 得到新的流形W'(作为配边看(W';W'_-,W_+')), 那么 W_+'和W_+ 之间就差了一个 k Surgery(下边界保持不变).
    • 简单来说, 在流形上粘 k+1 handle 就是在边界上做 Surgery.

4.2 Dehn Surgery

后面, 我们的 surgery 都指 dim=3 的 1-surgery. ## Definition

  • (Integer) Surgery
    • 回顾在 dim 3, S^3上做 Surgery. 我们需要在S^3上挖去一个 knot 的管状邻域, 然后用边界T^2=S^1\times S^1上的恒等映射将一个新的S^1\times D^2 粘回去.
    • 首先注意到, S^3 上挖掉 knot 的管状邻域之后一般不是 solid torus.
    • 粘回去 S^1\times D^2是一个 solid torus, 故粘接它可以被视为粘接h^2+h^3.
      • 我们知道, 在 dim=3 时, 粘接的 2-handle 会被曲面上简单闭曲线的 (同痕等价于同伦)同伦类唯一决定. 粘接 3-handle 的方式是唯一的.
      • 我们先关心这个 h^2, 只要找到简单闭曲线. 要知道 2-handle 粘到哪里, 只要知道哪个曲线会新 bound disk.
        • 根据 Surgery 的定义, 在公共边界 S^1\times S^1上看. 第二个S^1在S^3中 bound 一个 disk. 第一个S^1粘回去之后, 会 bound 一个 disk. 即第一个S^1 会新 bound 一个 ball.
        • 所以我们只要在 S^3中找到S^1\times S^1上的第一个S^1(它平行于纽结K, 落在管状邻域 NK 的边界上).
          • 这个平行于 K的曲线恰好就是描述 framing 的那一条. 当 framing=n 时, 它沿着\partial NK 的 longitude (由 Seifert framing 定下里的) 转 1 圈, 沿着 meridian 转 n 圈.
          • 在图上看:
    • 综上, Integer surgery 会被一个纽结 K和整数n 决定.

上述过程中, 我们本质上往一个 3 mfld (S^3-NK) 上粘接了一个 solid torus. 但是粘接的方式是固定的. 于是复杂度完全被 N K 嵌入的方式决定. 我们当然也可以假设 NK 是平凡嵌入的, 用粘接映射来描述这个复杂度, 即挖去一个标准的 (不扭曲的) solid torus, 然后沿着上图中的蓝线粘接 2-handle.

  • Dehn Surgery
    • 但是, 理论上, 我们未必一定要用上图中的方式, 即 K'=n\mu+\lambda来粘接 2-handle:T^2上的任意简单闭曲线都可以. 在同调上, 这对应于K'=p \mu+q\lambda, 其中 p,q互素. 那么此时, 这样的闭曲线可以被一个有理数\frac{p}{q}\in \mathbb{Q}\cup \infty 来确定.
    • 这种从 S^3 出发, 做有理手术得到闭的 3 mfld 的方式被称为 Dehn Surgery.
    • 注意到, 如果限制 q=1, 我们就回到了之前的情形, 此时 K'=p\mu+\lambda 就对应于一个整数, 这恰好还原了之前 integer surgery 的定义.

我们不仅可以对于 knot 做整的, 有理的 Surgery, 我们也可以对 link 做, 定义为依次对每个 knot 做 surgery (与做 surgery 的顺序无关). 对于 link L以及系数x_i\in \mathbb{Q}, 我们定义 S^3_L(x_i)为从S^3出发对 linkL 做 (有理) Dehn surgery 得到的闭 3 mfld.

4.3 Surgery and Closed 3 mfld

  • 定理 1(Lickorish–Wallace): Integer surgery 与 3 mfld.
    • 任意闭 3 mfld M, 存在 link L\subset S^3以及整数n_i, 使得 M\cong S^3_L(n_i)
    • 即任意闭 3 mfld 可以通过对 link 做整手术得到.
  • 定理 2(Rohlin)
    • 任意闭 3 mfld M, 存在单连通的 4 mfld W使得\partial W=M.
    • 这等价于: 任意闭 3 mfld M存在 4 mfldW使得\partial W=M. 即配边群 \Omega_3^{SO}=0. (下面的命题中, 我们用这条作为定义).
  • 命题: 上述两定理等价
  • 证明:
    • 1\Rightarrow2: 使用 handle attaching 和 (integer) surgery 之间的关系.
      • 定理 1 说明从 S^3到M只要做一些 1 surgery. 于是如果我们从D^4出发, 做一些 2-handle attaching (与 1 surgery 对应), 那么我得到的流形W=h^0+(h^2_i)_i求边界就得到M. 显然 W 是单连通的.
    • 2\Rightarrow1:
      • 根据定理 2, 存在 4 mfld W, \partial W=M. 要证定理 1, 我们只要找到一个只有 0,2-handle 的 4 mfld 使其 bound M.
      • 我们可以从 W出发, 得到一个W', 使得 \partial W'=\partial W(=M), 且 W' 上面只有 0,2 handle (即消去原来的 1,3 handle)
      • 我们对 W做 handle 分解. 注意, 现在我们只要保持边界不变就可以. 我们从W_1 (小于等于 1 的 handle 粘出来的) 出发.
      • W_1形如\#S^1\times D^3, 它的边界为 S^1\times S^2. 后续, 我们的粘接发生在这个边界上, 且最后得到的边界就是对这个边界做 surgery.
        • 注意到我们将 \#S^1\times D^3替换成\#D^2\times S^2 不会改变边界, 后者对应与粘接 2-handle, 它的 Kirby diagram 为一些 framing=0 的 unknot 的不交并.
        • 我们保持后面的粘接方式不变.
        • 此时, 我们得到了 W''它没有 1-handle, 且满足\partial W''=\partial W=M. 即我们总可以消去 1-handle, 用 2-handle 替代.
      • 我们对 W''做对偶, 得到一个配边(\overline{W''};M,S^3), 我们用上面的技术, 消去 \overline{W''}中的 1-handle (即原来W''中的 3-handle, 这个过程只会引入 2-handle.). 记得到的流形(配边)为(\overline{W'};M,S^3).
      • 最后, 再对偶一次, 我们得到流形 (W';S^3,M), 此时 W' 中只剩下 0,2 handle, 即为所求.

在上述证明中, 我们事实上也证明了定理 2 的两种表述等价.

  • Surgery 和 Dehn twist
    • 注意到在 3 mfld 上面做 Surgery 会保持原来的边界不变, surgery 会给出边界上曲线的重参数化, 从而诱导边界上的自同胚. 这个自同胚属于曲面的映射类群(…).
    • 记 K'是 solid torus 边界上的曲线, 将它向 solid torus 里面推得到 (边界的 collar nbhd 上面的曲线)K.
    • Claim: 沿着 K做+1 surgery 诱导的边界自同胚就是沿着K' 做 Dehn twist.
    • 证明: (下图对应顺时针的 dehn twist)
    • Remark:
      • 对 solid torus 沿着 core circle 做 surgery 得到的 3 mfld 仍然是 solid torus. 因为当我们在 solid torus 上挖去 core circle 的管状邻域后, 得到一个空心的 T^2\times I, 它可以被视为 collar neighborhood, 它的拓扑信息是平凡的. 重新粘接的结果只会依赖于粘回去的东西.
      • 更一般的, 对于 collar nbhd 上的曲线做 surgery 不会改变 collar nbhd, 从而诱导 collar 上的自同胚, 限制在边界上得到边界的自同胚.
      • 对于 solid torus 里的一般的位置的纽结 K, 做 surgery 之后得到的 3 mfld 未必还是 solid torus, 新的流形的边界(保持不变)仍然是 T^2. 此时 surgery 仍然可以诱导边界上的自同胚, 这个自同胚和 dehn twist 对应.

  • 定理 1 的证明: 使用 3 mfld 的 heegaard splitting.
    • 任意 3 mfld M, 存在 g, 使得 M=H_g\cup_{\phi} H_g. 注意到, S^3=H_0\cup H_0=H_g\cup_{\phi_0} H_g. 不难看出, \phi和\phi_0之间差了个MCG(\Sigma_g)中的元素, 记为f, 我们知道 f 可以被 dehn twist 生成.

定理 1 本质上给出了满射: \{ \text{framed link in } S^3 \}\twoheadrightarrow\{ \text{closed 3 mfld} \} 先映射到 2-handlebody, 然后取边界得到闭 3 mfld.

一个自然的问题是何时两组这样的 framed link, 给出相同的闭 3 mfld. 即上述映射的 “kernel” 是谁. 这一小节, 我们回答这个问题.

我们先给出答案: “kernel”由 kirby moves 生成, 即两个 framed link 等价, 当且仅当它们差了一系列的 kirby move.
